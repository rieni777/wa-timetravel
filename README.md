# wa-timetravel

**Read your WhatsApp chats like they're happening again.**

---

I built this because I wanted to see what we were talking about on this exact date — one year ago. Two years ago. Five.

My WhatsApp chat export was just a `.txt` file sitting on my phone and I didn't know what to do with it. Every third-party tool wanted me to upload it somewhere. I didn't want to upload it anywhere. These conversations are private. They belong to me.

So I built a viewer that runs entirely in your browser. Nothing leaves your device. Ever.

---

## What it does

- **Jump to any date** — pick a date, see exactly what was said that day. Or jump by year with one tap.
- **Search across everything** — find any word, any memory, across years of messages. Navigate between results one by one.
- **Works offline** — open the HTML file, load your `.txt` export, done. No internet needed after that.
- **Save as PDF or HTML** — keep a copy that doesn't depend on WhatsApp being around.
- **Works with any WhatsApp export** — Android, iPhone, group chats, personal chats.

## How to use

1. Export your WhatsApp chat: open a chat → ⋮ menu → **More** → **Export chat** → **Without media** (or with media, both work)
2. Download or clone this repo
3. Open `index.html` in your browser
4. Click **Open the app** → load your `.txt` file
5. That's it.

## Privacy

This tool runs 100% in your browser. Your chat file is never uploaded anywhere. There are no servers, no accounts, no tracking. The code is open source — you can read every line of it.

## Contributing

Pull requests welcome. If you have a feature idea or found a bug, open an issue.

# wa-timetravel

**Baca chat WhatsApp-mu seperti sedang terjadi lagi.**

---

Aku bikin ini karena aku ingin lihat — kita lagi ngomongin apa ya, di tanggal yang sama, setahun lalu? Dua tahun lalu? Lima tahun lalu?

File ekspor WhatsApp-ku cuma file `.txt` yang nganggur di HP. Setiap tool yang ada minta aku upload ke server mereka. Aku nggak mau. Percakapan ini privat. Ini milikku.

Jadi aku bikin viewer yang jalan sepenuhnya di browser. Tidak ada data yang keluar dari perangkatmu. Selamanya.

---

## Fitur

- **Lompat ke tanggal tertentu** — pilih tanggal, lihat apa yang dibicarakan hari itu. Atau lompat per tahun dengan satu klik.
- **Cari kata apa saja** — temukan kata, momen, atau memori apa pun dari seluruh riwayat chat. Navigasi antar hasil satu per satu.
- **Jalan offline** — buka file HTML-nya, load file `.txt` ekspor-mu, selesai. Tidak butuh internet setelah itu.
- **Simpan sebagai PDF atau HTML** — punya salinan yang tidak bergantung pada WhatsApp.
- **Kompatibel dengan semua ekspor WhatsApp** — Android, iPhone, grup, personal.

## Cara pakai

1. Ekspor chat WhatsApp-mu: buka chat → menu ⋮ → **Lainnya** → **Ekspor chat** → **Tanpa media** (atau dengan media, keduanya bisa)
2. Download atau clone repo ini
3. Buka `index.html` di browser
4. Klik **Open the app** → load file `.txt`-mu
5. Selesai.

## Privasi

Tool ini berjalan 100% di browser-mu. File chat-mu tidak pernah dikirim ke mana pun. Tidak ada server, tidak ada akun, tidak ada tracking. Kodenya open source — kamu bisa baca setiap barisnya.
